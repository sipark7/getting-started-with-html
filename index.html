"Why Computers are Stupid"

Computers are tools that takes instructions literally. This makes them very unforgiving since a small mistake by a programmer can cause huge problems in a program.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box"

Inline
bold - b content /b
emphasis - em content /em
anchor - a content /a
images - img src = "url" alt = "text" (void tag)
tag - tag attr = "text" /tag
line break - br (void tag)

Block - "Invisible Box"
paragraph - p content /p
section - div content /div
